RearSilver's Auto Start

The RearSilver Auto Start script helps you automatically launch and close your essential streaming applications alongside OBS Studio.
Configure a list of your favourite programs (like chatbots, music players, lighting controls, etc.), and this script will launch them when OBS starts and (attempts to) close them when OBS exits.
You can also manually launch or close these programs using buttons in the script's UI or even trigger the launch with a dedicated hotkey! Get your stream ready with a single action.

  • Auto-Launch on OBS Start:
    Automatically launch your specified programs when you open OBS Studio.
  • Graceful Close on OBS Exit:
    Attempts to close the launched programs when you close OBS.
  • Manual Launch & Close Buttons:
    Provides "Launch Now" and "Close Now" buttons in the script UI for on-demand control.
  • Optional Hotkey Trigger:
    Set up a hotkey in OBS to manually launch your configured programs whenever you need them.
  • Run as Administrator Option:
    Allows you to specify if a program should be launched with administrator privileges
    (use with caution and ensure OBS is also run as admin for best results).
  • Configurable Program List:
    Add up to 10 different programs with their executable paths and optional launch arguments.
  • Saves Configuration:
    Your program list and settings are saved between OBS sessions.

Setup Instructions:

  1. Download the Rearsilver Auto Start.lua file.
  2. Move it to your OBS Studio Scripts folder.(if you don't have a scripts folder, create one) and extract it.
  3. In OBS Studio, go to Tools > Scripts.
  4. Click the "+" button in the Scripts window and add the Rearsilver Auto Start.lua file.
  5. In the script's UI, you will see a list of program configurations. For each program you want to manage:
  6. Click the "Browse" button to select the executable file (.exe on Windows) of the program.
  7. Optionally, enter any "Launch Arguments" the program might require.
  8. Toggle the "Run as Administrator" checkbox if the program needs elevated privileges.
  9. Check the "Auto-launch programs on OBS start" box if you want the programs to launch automatically when OBS opens.
  10. Alternatively, check the "Use hotkey instead of auto-launch" box if you prefer to trigger the launch manually using a hotkey. If you select this, you'll need to assign a hotkey in OBS Settings > Hotkeys under the "Launch Programs (RearSilver)" action.
  11. Click the "Launch Now" button to manually start all configured programs.
  12. Click the "Close Now" button to attempt to close all programs launched by the script.

The RearSilver Browser Refresh script is here to save the day!

No more fumbling through source properties or restarting OBS! Keep your overlays, widgets, and web elements running smoothly with this simple yet effective script.

    Keep your focus on your content, not on troubleshooting technical hiccups with your browser sources.

  4. Click the "+" button in the Scripts window and add the RearSilver Browser Refresh.lua file.

  6. In the script's UI, you will see a dropdown menu listing all your current browser sources in your active scene.

RearSilver's Quick Text Source Creator

The RearSilver Text Source script provides a dynamic and convenient way to control the text displayed in a specific Text (GDI+) source within your OBS scene.
Instead of constantly editing the text source properties, this script gives you a simple text input field in its UI where you can type or paste the desired text, and it will instantly update in your stream.
Perfect for displaying quick messages, shoutouts, live reactions, or any other temporary text elements!

  • Real-time Text Updates:
    Change the text displayed in your chosen Text (GDI+) source directly from the script's UI, with instant updates on your stream.
  • Easy Text Input:
    A simple text box allows for quick typing or pasting of the desired message.
  • Source Selection:
    A dropdown menu lets you select which of your existing Text (GDI+) sources you want to control with the script.
  • Dynamic On-Screen Information:
    Display temporary information or engage with your audience in real-time without interrupting your stream flow.
  • Saves Configuration:
    Your program list and settings are saved between OBS sessions.

Setup Instructions:

  1. Download the Rearsilver Auto Start.lua file.
  2. Move it to your OBS Studio Scripts folder.(if you don't have a scripts folder, create one) and extract it.
  3. In OBS Studio, go to Tools > Scripts.
  4. Click the "+" button in the Scripts window and add the RearSilver Text Source.lua file.
  5. In the script's UI, you will see a dropdown menu listing all your current Text (GDI+) sources in your active scene.
  6. Select the Text (GDI+) source you want to control.
  7. Type or paste the text you want to display into the provided text field in the script's UI. The text in your selected source will update immediately.

RearSilver's Mighty Muted Mic Reminder/Notice

You ever have those moments where you're chatting away on stream, only to realize you've been muted the whole time? Don't worry, this musky Husky has your back!
Never let a muted microphone ruin your stream again!
RearSilver's Mighty Muted Reminder is a simple yet powerful OBS script that helps you stay aware of your microphone's mute status.
By automatically showing a customizable text indicator when your mic is muted, this script ensures you'll always know when your audience can (or can't) hear you.
It's easy to set up, works across multiple scenes, and gives you full control over the styling of the text indicator.

  • Automatic Mute Detection:
    Displays a text indicator when your selected microphone is muted and hides it when unmuted.
  • Customizable Text Source:
    Use any "Text (GDI+)" source in OBS to style the mute indicator exactly how you want.
  • Scene Flexibility:
    Works across all scenes where the text source is present, so you don't need to configure it for each scene individually.
  • Simple Setup:
    Just select your microphone and text source in the script's settings, and you're good to go.
  • Set It and Forget It:
    Once configured, the script runs automatically in the background, keeping you informed without any extra effort.
  • Streamer-Friendly Design:
    Created by a streamer for streamers, this script is designed to solve a common problem with minimal hassle.
  • Saves Configuration:
    Your program list and settings are saved between OBS sessions.

Setup Instructions:

  1. Download the RearSilver's Mighty Muted Reminder.lua file.
  2. Move it to your OBS Studio Scripts folder.(if you don't have a scripts folder, create one) and extract it.
  3. In OBS Studio, go to Tools > Scripts.
  4. Click the "+" button in the Scripts window and add the Rearsilver Mighty Muted Reminder.lua file.
  5. In OBS, create a "Text (GDI+)" source in the scene(s) where you want the mute indicator to appear.
  6. Customize the text source's appearance (e.g., font, size, color, etc.) to your liking.
  7. Select your microphone input from the dropdown list.
  8. Enter the exact, case-sensitive name of the "Text (GDI+)" source you created.
  9. Follow the on-screen instructions in the script's UI for additional guidance.
  10. Alternatively, check the "Use hotkey instead of auto-launch" box if you prefer to trigger the launch manually using a hotkey. If you select this, you'll need to assign a hotkey in OBS Settings > Hotkeys under the "Launch Programs (RearSilver)" action.
  11. Mute and unmute your microphone to ensure the text indicator appears and disappears as expected.
  12. Copy and paste the text source into other scenes where you want the mute indicator to appear.
