Part 1: Moderator Guide

General Expectations (The Core Rules)

  1. You Represent the Channel: You are an extension of RearSilver. Ban with kindness (if possible), but bully with humour. Your tone sets the standard.
  2. Use Common Sense: If it’s hateful, harassing, spam, or disrupts chat flow (even if not technically rule-breaking), act fast. Don't let a bad vibe linger.
  3. Trust is Key: Mods are trusted to use judgement — no need to ask for permission before moderating. Act, then inform.
  4. Communicate with Each Other: If you action something serious, drop a quick note in the mod chat/Discord. This prevents duplicate actions.
  5. Confidentiality: Don’t leak mod-only info (our internal tools, systems, or stream private chat).
  6. The Humour Filter: Keep the humour dark but not cruel. Our content is adult and explicit, but our community is inclusive.
  7. If in doubt: Timeout first, ask later. Better to cool someone down with a 600 second time-out than risk escalation and a ban that derails the stream.
  8. Set the trend of commands. Please set the trend of clips and quotes. If people don't know to do it, they won't. Encourage them to use the commands.

Moderator Philosophy (Chaos, not Cruelty)

“If you wouldn’t want it shouted across a pub, don’t let it live in chat.”
  • Prioritise Safety: We lean toward chaos, not cruelty. Let fun breathe but stop things before they sour.
  • Inclusivity is Non-Negotiable: We are an LGBTQ+ safe space first, entertainment second. Zero tolerance for bigotry of any kind.
  • Handle Appeals Privately: Do not argue with bans or timeouts in public chat. Direct users to the appropriate Discord channel.
  • FrontSilver Trusts You: Don’t make him regret it.

Discord Server (Mod Communication Hub)

Use the server channel to discuss all mod actions and situations. Also, make friends with the other mods, they are your allies.

  • Channel Link: #twitch-moderator
  • Stream Log: Use this to see all Mod actions taken in chat. #stream-log
  • Privacy: This is a safe channel to discuss people and conduct. These discussions could get personal, so please keep it private.
  • Threads: General conversation is promoted here but for specific situations, please create a thread to keep things organised.

Emergency & Raid Protocol

Use this protocol for overwhelming spam or hate raids:

  1. Assess: Is the situation manageable, or is the spam/hate overwhelming?
  2. Initiate Lockdown: If overwhelming, use *!emergency. This activates slow mode, followers-only, and clears chat.
  3. Monitor & Report: While in lockdown, continue banning offenders. Inform other mods in Discord.
  4. Stand Down: Once the wave has passed, use *!emergencyover to return chat to normal.

Common Mod Action Scenarios

Offence Suggested Action Contextual Notes
Spam / Flooding 30's –300's Timeout Usually a bot or new user. Use *!clear if it floods the screen.
Aggressive / Rude 600's – 1200's Timeout For personal attacks, back-seat gaming, or excessive negativity.
Unsolicited Links Remove Link + Warn / 300's TO Bot handles most. Use *!lpallow to approve a link manually.
Bigotry / Hate Speech PERMANENT BAN Zero-tolerance. Use *!ban. Inform mod chat immediately.
Repeat Offenders Escalate to BAN If a user is timed out multiple times, escalate to a ban.
