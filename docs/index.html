🧰 About the Tools

Simple tools to solve real streamer problems.

This toolkit isn’t a bloated app—it’s a lightweight collection of stream-specific utilities that help with everyday tasks:

  • 📝 Stream Title Generator - Generate creative and engaging Twitch stream titles & go live notifications made to share quickly and easily.
  • 📒 Twitch Story Generator – Created for Twitch streamers to easily generate story images for Twitch scheduled stream, raid thanker, subscriber/gifted sub thanker and a new follower.
  • 📅 Schedule Builder – Quickly create stream schedules with exportable visuals and time zone conversions.
  • 💻 OBS Scripts – Pre-made scripts designed to make automation of streaming easier.
  • 🕒 Discord Timestamp Generator – Created to generate a Discord timestamp that will show the local time version of your input depending on who is viewing it.
  • 🌈 Discord Coloured Text Generator - Create colourful messages for Discord using ANSI colour codes

More tools are always in the works based on community feedback.
